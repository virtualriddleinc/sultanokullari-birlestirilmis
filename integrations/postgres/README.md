# PostgreSQL

Yerel geliştirme için kök dizindeki `docker-compose.yml` kullanılır.

Üretimde Supabase veya yönetilen PostgreSQL kullanılacaksa `DATABASE_URL` ortam değişkenini ilgili bağlantı dizesi ile ayarlayın. Proje kurallarına göre Supabase entegrasyonu gerektiğinde `integrations/supabase` altında yapılandırılabilir.
